Scripts created as a part of Honours Research Programme (Maastricht Science Programme, Maastricht University) 
Conducted at Nikhef, as a part of Tracking Optimisation Project
Scripts are intented be used inside ACTS framework/repository to generate plots for tracking efficiency analysis.
ROOT is a framework used for generating histograms
Scripts are written in Python, but similar logic can be followed in C++ 
Example plots are also given
